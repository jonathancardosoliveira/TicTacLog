# Jogo da Velha em Prolog

Implementação completa do **Jogo da Velha** (Tic‑Tac‑Toe) em Prolog, desenvolvida como trabalho prático para a disciplina de Paradigmas de Programação. 
<!-- O jogo segue todas as regras clássicas, possui interface interativa via teclado e, opcionalmente, um adversário controlado por inteligência artificial utilizando o algoritmo **MinMax**. -->

---

## 📖 Sobre o projeto

O programa implementa o clássico jogo da velha em **Prolog**, utilizando fatos, regras e recursão para controlar o fluxo da partida.  
Dois jogadores se alternam, inserindo coordenadas linha/coluna para marcar suas peças. O tabuleiro é exibido após cada jogada, e o jogo detecta automaticamente vitória, empate ou jogadas inválidas.

---

## 🚀 Como executar

1. Tenha o **SWI-Prolog** instalado em seu sistema.  
   - [Download oficial](https://www.swi-prolog.org/download/stable)

2. Clone o repositório:
   ```bash
   git clone https://github.com/jonathancardosoliveira/TicTacLog
   cd jogo-da-velha-prolog

3. Inicie o SWI-Prolog e carregue o arquivo principal.

4. Inicie o jogo com o predicado:
   " ?- iniciar. "

5. E divirtase jogando, Para sair a qualquer momento, digite -1,-1.
